Про компанію

Компанія Травмамед - український дистриб’ютор медичних імплантів та інструментів для ортопедії, травматології, нейрохірургії та ЩЛХ.

Асортимент товарів включає:

  • Системи шунтування та дренажі для СМР
  • Синтетичний кістковий замінник
  • Імпланти для травматології та ортопедії (інтрамедулярні стрижні, пластини, гвинти, АЗФ, анкери, ендобаттони)
  • Ендопротези для кульшового, ліктьового та плечового суглобу
  • Імпланти для нейрохірургії (кейджи, транспедикулярні системи, краніопластика, кіфопластика, вертебропластика)
  • Імпланти для ЩЛХ
  • Системи для промивання ран та ВАК терапії
  • Інструменти для установки та видалення

Компанія більше 7 років є ексклюзивним постачальником світових брендів

  • Bıçakcılar (Туреччина)
  • Bioceramed Portugal (Португалія)
  • Jiangsu Jinlu Medical Device (Китай)
  • Lider Medica (Італія)
  • Orthotech (Індія)
